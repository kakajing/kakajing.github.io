发表于 2017-01-16 | 分类于 spring

Installation or Setup

第一次使用Spring Boot进行设置是相当快的，这要归功于Spring Community的辛勤工作。

先决条件：

  1. Java安装
  2. Java IDE推荐不需要（Intellij，Eclipse，Netbeans等）

你不需要安装Maven和/或Gradle。 Spring Initializr 生成的项目带有一个Maven包装器（命令mvnw）或Gradle包装器（命令gradlew）。

打开您的Web浏览器到https://start.spring.io这是一个启动板，用于创建新的Spring Boot应用程序，现在我们将以最低限度使用。

如果这是您首选的构建工具，请随意从Maven切换到Gradle。

在“搜索依赖关系”下搜索“Web”并添加。

单击Generate Project！

这将下载一个zip文件名为demo Feel免费提取这个文件，无论你想在你的计算机上。

如果选择maven，请导航到基本目录的命令提示符，并发出mvn clean install命令

你应该得到一个构建成功输出：

运行应用程序：mvn spring-boot:run

现在你的Spring Boot应用程序启动。 将您的Web浏览器导航到localhost:8080

恭喜！ 你刚刚得到你的第一个Spring启动应用程序并运行。 现在让我们添加一小段代码，这样你就可以看到它的工作。

因此，使用ctrl + c退出当前运行的服务器。

导航到：src/main/java/com/example/DemoApplication.java更新此类以具有控制器

现在允许再次构建和运行项目mvn clean install spring-boot:run！

现在浏览您的web浏览器到localhost:8080

Hello World!

恭喜！ 我们刚刚完成创建一个Spring Boot应用程序，并设置我们的第一个控制器返回“Hello World!“ 欢迎来到Spring Boot的世界！
